Module nodejs

The nodejs module contains support for running Node.js applications from Qbs.

General Properties

applicationFile

Type:path
Default:undefined

The input JavaScript file whose corresponding output will be executed when running the Node.js application. If this property is undefined, the product will not be runnable.
